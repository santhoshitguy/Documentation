Installation

Setup the development environment

  1. Install Node and npm latest version if they are not available in your machine.
  2. Angular CLI is the tool for latest angular application setup. Install it before start to setup the application.
  3. Check the node and npm version using following command: node --version && npm --version

    If the local machine has Angular CLI already, then we need to update the version to confirm the application is running without any hassels.

    npm uninstall --global @angular/cli

    npm install --global @angular/cli@latest

Angular Quickstart

  1. To start and run the theme, change the directory to [anglar]/anglar/

  2. Install the angular dependencies by using the following commands.

    • npm clear cache --force - This will remove the old cache of any node-modules in the same folder
    • npm install - This command installs all the dependencies in the node_modules folder based on the lists in the package.json file. By default, npm install will install all modules listed as dependencies in package.json.
    • ng serve or npm start - This will starts compilation and launch the application in the default port of local machine as open it in the browser after compilation completes.

For more detailed information to install the Angular application, visit this official Angular documentation website.

Polyfills

Angualr is built and compile on the latest web standard and technologies. It supports mostly all kind of browsers available but not all the funcitionalities. Every browser need atleast a single polyfills to run the application.

To support Angular in IE, open this file [anglar]/anglar/src/polyfills.ts Uncomment all the imports under Browser Polyfills list and classlist.js

To read more about the polyfills, please click here. https://angular.io/guide/browser-support
