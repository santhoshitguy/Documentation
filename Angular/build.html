Build Tools

Installing Tools

  • The following steps are required to setup and run the theme. If these steps are not enough then do some research on how to start and run a application.
    • First step we need to install the node and npm latest version. Go to Node JS
    • Download and install GIT for your platform from http://git-scm.com/downloads
    • To confirm node and npm installs correctly, run the following command. node --version && npm --version
  • After install the above tools
    • Navigate to the theme folder and then open the theme in any of your favorite editor.
    • Open the theme in Visual Studio code/terminal and run the following command npm install -g @angular/cli to install the angular command line interface and the application dependencies.
    • To run and compile the application in local machine use the ng serve or npm start command.
  • If your machine has the latest node and npm, then open your browser and browse http://localhost:4200

Building for Production

After complete the development we need to deploy the application on the any server to make it available for end user access.

When we make final files, we need to ignore all sourcemaps and other chunk files as well. Run the following command to generate deployable build.

ng build --prod --aot or npm run build --prod --aot

When we use the flag "--aot" makes the compilation process too long because some internal functionalities. So please be patience when make the final AOT enabled build.

After the command completes without throws any error. It generates the folder 'dist' which ready to place in the server and can be accessible by the end users.
